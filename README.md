# Contact Book

A beginner-friendly side project built with easy steps, growing into a full-stack app.

## Features

- Add a new contact (name, phone, email)
- View all contacts
- Search for a contact by name
- Delete a contact

## How to Run

Make sure you have Python installed, then run:

```bash
python contact_book.py
```
